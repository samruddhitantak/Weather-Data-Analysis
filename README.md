# 🌦️ Weather Data Analysis using Python

## 📌 Project Overview

This project analyzes weather data using Python. It involves cleaning the dataset, exploring weather patterns, and visualizing trends in temperature, humidity, precipitation, and wind speed.

---

## 🎯 Objectives

- Load and clean weather data
- Handle missing values
- Perform Exploratory Data Analysis (EDA)
- Analyze temperature, humidity, and rainfall trends
- Compare monthly weather conditions
- Generate visualizations
- Save the cleaned dataset

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📂 Dataset Columns

- Location
- Date_Time
- Temperature_C
- Humidity_pct
- Precipitation_mm
- Wind_Speed_kmh

---

## 📊 Visualizations

- Temperature Trend
- Rainfall Trend
- Monthly Average Temperature
- Monthly Average Humidity
- Correlation Heatmap
- Temperature Boxplot
- Temperature vs Humidity Scatter Plot
- Monthly Weather Comparison

---

## 📈 Key Insights

- Identified monthly temperature trends.
- Compared humidity across months.
- Analyzed rainfall patterns.
- Observed relationships between weather variables.
- Explored correlations using a heatmap.
